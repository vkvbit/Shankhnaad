Coding League








Introduction


Coding League is a 2 hours long coding contest where you have to solve 4 questions related to basic problem solving.



Rules & Regulations


  1. This contest is for individuals.
  2. Discussing problems or any aspect of problem, on any other platform on web, on identification, could lead to disabling of respective account and banning from the community.
  3. All programming languages are allowed.
  4. Each program will be tested based on our critical test data. However, output and input should be exactly as specified in the samples provided.
  5. There will be 4 questions in the contest.
  6. The candidate must solve the questions within 2 hours.



Criteria of Judgement


  1. Each challenge has a pre-determined score.
  2. Your score for a problem depends on the number of test cases your submission successfully passes.
  3. A participant's total score is the sum of the scores earned for each problem attempted. If you submitted more than one solution for a problem, only your highest score achieved will be used in this calculation.
  4. Participants are ranked by score, with the cumulative time taken (between the contest's start time and the time of your correct submission) used to break ties.
